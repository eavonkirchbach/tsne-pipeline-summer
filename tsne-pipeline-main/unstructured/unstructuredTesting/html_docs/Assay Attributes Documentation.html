Assay Attributes Documentation

The wizard for the creation of Assay Runs imports files with assay data and attempts to map their assay measurements to legal Measurement Attribute.

The following section lists for each Assay the recognized attributes, their units, and other useful information. Please note that the set of Measurement Attribute shown for each Assay represents the types of measurements that can be stored in the database for that Assay.

If you cannot find a Measurement Attribute you would like to use, please contact an administrator.
